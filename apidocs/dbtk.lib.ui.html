Module d.l.ui

Part of dbtk.lib

Database Toolkit UI

This module contains the UI elements of the database toolkit platform.

This module should not be run directly; instead, individual scripts, when run, should run the launch_wizard function.

Functionlaunch_wizard

Launches the download wizard.

Launches the download wizard.
