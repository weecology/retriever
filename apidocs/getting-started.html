Database Toolkit - Getting Started

Windows

A windows executable is provided. Simply download and run dbtk.exe.

Linux

For Debian-based Linux distributions (including Ubuntu), a Debian package, python-dbtk_1.0-1_all.deb, is provided.

  1. Open the package with Synaptic Package Manager or another Debian package management application and install it.
  2. From the terminal, type dbtk to launch the Database Toolkit. You should run this command from the directory you want to store configuration files and raw data files in, so you may want to create a new directory.

Source

To install the Database Toolkit from source, you'll need Python 2.6 with the following packages installed:

  • wxPython
  • xlrd

The following packages are optional:

  • MySQLdb (for MySQL)
  • sqlite3 (for SQLite)
  • psycopg2 (for PostgreSQL)
  • pyodbc (for MS Access - this option is only available on Windows)

To install from source:

  1. Download the dbtk source bundle and extract it onto your machine.
  2. From the directory containing setup.py, run the following command: python setup.py install
  3. After installing, type dbtk from a command prompt to launch the Database Toolkit, or find dbtk/main.py on your machine and run it in Python.
