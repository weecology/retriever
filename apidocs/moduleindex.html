Package dbtk

  • dbtk.lib -

    dbtk.lib contains the core Database Toolkit modules.

    • dbtk.lib.engines -

      Database Toolkit Engines

    • dbtk.lib.excel -

      Database Toolkit Excel Functions

    • dbtk.lib.models -

      Database Toolkit Data Model

    • dbtk.lib.tools -

      Database Toolkit Tools

    • dbtk.lib.ui -

      Database Toolkit UI

    • dbtk.lib.ui_pages -

      Classes representing pages for the DBTK wizard.

  • dbtk.main -

    Database Toolkit Wizard

  • dbtk.scripts -

    dbtk.scripts contains scripts representing individual datasets.

    • dbtk.scripts.CRC_avianbodymass -

      Database Toolkit for CRC Handbook of Avian Bird Masses companion CD

    • dbtk.scripts.EA_avianbodysize2007 -

      Database Toolkit for Avian Body Size

    • dbtk.scripts.EA_ernest2003 -

      Database Toolkit for Mammalian Life History Database

    • dbtk.scripts.EA_pantheria -

      Database Toolkit for Pantheria dataset

    • dbtk.scripts.EA_portal_mammals -

      Database Toolkit for the Portal Project mammals

    • dbtk.scripts.all -

      Importing MODULE_LIST from this module gives a list of all dataset scripts.

    • dbtk.scripts.bbs -

      Database Toolkit for Breeding Bird Survey

    • dbtk.scripts.gentry -

      Database Toolkit for Alwyn H. Gentry Forest Transact Dataset

  • dbtk.setup -

    Use the following command to install dbtk: python setup.py install

  • dbtk.tests -

    Database Toolkit Tests
